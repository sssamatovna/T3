# Т3: Составление тест-плана

## Общая информация 

**API:** GET /products/{productId}/cost\
**Параметры:**\
- authToken - строка 16 символов(цифры и буквы)\
- start - индекс количества приобретенных продуктов\
- count - количество товаров для расчета

---

## Тест-кейсы

| ID        | Описание                                                                                       | Шаги                                                                                                            | Ожидаемый результат       |
|:----------|:-----------------------------------------------------------------------------------------------|:----------------------------------------------------------------------------------------------------------------|:--------------------------|
| **TC-01** | Расчет стоимости: один товар. Предусловие: Валидный authToken (16 символов)                    | 1. Выполнить запрос `/products/123/cost?authToken=1234567890123456&count=1&start=0`<br/>2.Получить ответ        | 200 OK<br/>`{"cost":30}`  |
| **TC-02** | Расчет стоимости: пять товаров. Предусловие: Валидный authToken (16 символов)                  | 1. Выполнить запрос `/products/123/cost?authToken=1234567890123456&count=5&start=0`<br/>2.Получить ответ        | 200 OK<br/>`{"cost":100}` |
| **TC-03** | Расчет стоимости при start=2,count=2. Предусловие: Валидный authToken (16 символов)            | 1. Выполнить запрос `/products/123/cost?authToken=1234567890123456&count=2&start=2`<br/>2.Получить ответ        | 200 OK<br/>`{"cost":35}`  |
| **TC-04** | Расчет стоимости при превышении размера партии. Предусловие: Валидный authToken (16 символов)  | 1. Выполнить запрос `/products/123/cost?authToken=1234567890123456&count=7&start=0`<br/>2.Получить ответ        | 200 OK<br/>`{"cost":155}` |
| **TC-05** | Расчет стоимости при нулевом количестве товаров. Предусловие: Валидный authToken (16 символов) | 1. Выполнить запрос `/products/123/cost?authToken=1234567890123456&count=0&start=0`<br/>2.Получить ответ        | 200 OK<br/>`{"cost":0}`   |
| **TC-06** | Без authToken                                                                                  | 1. Выполнить запрос `/products/123/cost?count=2&start=2`<br/>2.Получить ответ                                   | 401 Unauthorized          |
| **TC-07** | Неверная длина токена                                                                          | 1. Выполнить запрос `/products/123/cost?authToken=123&count=2&start=2`<br/>2.Получить ответ                     | 401 Unauthorized          |
| **TC-08** | Невалидный токен                                                                               | 1. Выполнить запрос `/products/123/cost?authToken=12?4!"№90%:?*5#6&count=2&start=2`<br/>2.Получить ответ        | 401 Unauthorized          |
| **TC-09** | Несуществующий productId. Предусловие: Валидный authToken (16 символов)                        | 1. Выполнить запрос `/products/999999/cost?authToken=1234567890123456&count=0&start=0`<br/>2.Получить ответ     | 404 Not Found             |
| **TC-10** | Отрицательный count. Предусловие: Валидный authToken (16 символов)                             | 1. Выполнить запрос `/products/123/cost?authToken=1234567890123456&count=-5&start=2`<br/>2.Получить ответ       | 400 Bad Request           |
| **TC-11** | Отрицательный start. Предусловие: Валидный authToken (16 символов)                             | 1. Выполнить запрос `/products/999999/cost?authToken=1234567890123456&count=0&start=0`<br/>2.Получить ответ     | 404 Not Found             |
| **TC-12** | Значение start - строка. Предусловие: Валидный authToken (16 символов)                         | 1. Выполнить запрос `/products/123/cost?authToken=1234567890123456&count=0&start=abc`<br/>2.Получить ответ      | 400 Bad Request           |
| **TC-13** | Неверный HTTP-запрос (POST). Предусловие: Валидный authToken (16 символов)                     | 1. Выполнить запрос `POST /products/123/cost?authToken=1234567890123456&count=0&start=abc`<br/>2.Получить ответ | 405 Method Not Allowed    |
